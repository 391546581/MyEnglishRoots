# RootFlow 快速使用指南

## 🚀 快速开始

### 1. 标记高频词汇等级

运行以下命令为常见的 IELTS/SAT/GRE 词汇添加等级标签：

\`\`\`bash
npx tsx scripts/update-word-levels.ts
\`\`\`

这将标记：
- **200+** IELTS 词汇
- **300+** SAT 词汇  
- **500+** GRE 词汇

### 2. 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

访问 [http://localhost:3000](http://localhost:3000)

## 📖 功能使用

### 搜索单词

1. 在首页搜索框输入单词或词根（如 `ped`）
2. 查看匹配的词根和单词列表
3. 使用顶部的筛选按钮过滤特定考试词汇

### 查看单词详情

1. 点击任意单词进入详情页
2. **左侧面板**：查看词源、词根关联、同族单词
3. **右侧面板**：自动显示 LanGeek 词典信息
   - 大图展示
   - 中文翻译
   - 英文定义
   - 例句
   - 点击按钮跳转到完整词条

### 标记学习进度

- 点击单词卡片上的状态按钮循环切换：
  - 🔴 **NEW** → 🟡 **LEARNING** → 🟢 **MASTERED**

### 词汇筛选

在搜索结果页面，使用顶部筛选按钮：
- **All**: 显示所有结果
- **IELTS**: 仅显示雅思词汇
- **SAT**: 仅显示 SAT 词汇
- **GRE**: 仅显示 GRE 词汇
- **TOEFL**: 仅显示托福词汇

## 💡 高级功能

### 添加更多词汇等级

编辑 `scripts/update-word-levels.ts`，添加您自己的词汇列表：

\`\`\`typescript
const myCustomWords = ['word1', 'word2', 'word3'];

await prisma.word.updateMany({
  where: { text: { in: myCustomWords } },
  data: { level: 'CUSTOM_LEVEL' }
});
\`\`\`

### 导出学习进度

\`\`\`bash
# 查看已掌握的单词
npx prisma studio
# 在 Word 表中筛选 familiarity = 'MASTERED'
\`\`\`

### 批量导入词汇

参考 `docs/词汇等级标记指南.md` 了解如何从外部数据源导入词汇等级。

## 🎯 学习建议

1. **每天学习 10-20 个新词根**
   - 使用首页的"Shuffle"按钮发现新词根
   
2. **重点关注高频词汇**
   - 使用 IELTS/SAT 筛选器专注于考试词汇
   
3. **利用词根记忆**
   - 点击词根查看所有相关单词
   - 理解词根含义后，记忆同族单词更轻松
   
4. **定期复习**
   - 标记为 LEARNING 的单词需要定期回顾
   - 使用搜索功能快速找到需要复习的单词

## 🐛 常见问题

### Q: LanGeek 显示加载失败？
A: 检查网络连接，LanGeek API 可能有访问限制。可以点击"View Full Entry"按钮在新标签页打开。

### Q: 筛选器没有结果？
A: 确保已运行 `update-word-levels.ts` 脚本标记词汇等级。

### Q: 如何重置学习进度？
A: 使用 Prisma Studio 批量更新 `familiarity` 字段为 'NEW'。
