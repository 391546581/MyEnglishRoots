# 词汇等级标记指南

## 📚 如何为单词添加等级标签

目前数据库中的单词 `level` 字段默认为 `N/A`。要启用 IELTS/SAT/GRE 等筛选功能，您需要为单词添加等级标签。

### 方法 1: 手动更新（适合少量单词）

使用 Prisma Studio 可视化编辑：

\`\`\`bash
npx prisma studio
\`\`\`

然后在 Word 表中手动编辑 `level` 字段。

### 方法 2: 批量导入（推荐）

创建一个脚本来批量更新单词等级。以下是示例：

\`\`\`typescript
// scripts/update-word-levels.ts
import { PrismaClient } from '@prisma/client';

const prisma = new PrismaClient();

// 示例：IELTS 高频词汇列表
const ieltsWords = [
  'abandon', 'abstract', 'academic', 'accelerate', 'access',
  'accommodate', 'accompany', 'accomplish', 'accord', 'account',
  // ... 添加更多单词
];

// SAT 词汇列表
const satWords = [
  'abate', 'aberration', 'abhor', 'abscond', 'abstemious',
  // ... 添加更多单词
];

async function updateLevels() {
  // 更新 IELTS 词汇
  await prisma.word.updateMany({
    where: { text: { in: ieltsWords } },
    data: { level: 'IELTS' }
  });

  // 更新 SAT 词汇
  await prisma.word.updateMany({
    where: { text: { in: satWords } },
    data: { level: 'SAT' }
  });

  console.log('Updated word levels!');
}

updateLevels()
  .catch(console.error)
  .finally(() => prisma.$disconnect());
\`\`\`

运行脚本：
\`\`\`bash
npx tsx scripts/update-word-levels.ts
\`\`\`

### 方法 3: 从外部数据源导入

您可以从以下来源获取词汇等级数据：

1. **IELTS 词汇表**
   - 下载 IELTS 官方词汇列表
   - 解析 JSON/CSV 格式
   - 批量导入到数据库

2. **SAT/GRE 词汇表**
   - 使用开源词汇表（如 Quizlet、Anki 卡组）
   - 转换为 JSON 格式
   - 批量更新

3. **API 集成**
   - 某些词典 API 提供 CEFR 等级（A1-C2）
   - 可以映射到 IELTS/SAT 等级

## 🎯 推荐的等级标准

| 等级 | 适用范围 | 词汇量 |
|------|---------|--------|
| IELTS | 雅思考试 | ~8,000 |
| SAT | 美国大学入学考试 | ~3,500 |
| GRE | 研究生入学考试 | ~3,000 |
| TOEFL | 托福考试 | ~8,000 |

## 📝 示例数据

创建 \`data/word-levels.json\`：

\`\`\`json
{
  "IELTS": [
    "abandon", "abstract", "academic", "accelerate"
  ],
  "SAT": [
    "abate", "aberration", "abhor", "abscond"
  ],
  "GRE": [
    "abstruse", "accolade", "acerbic", "admonish"
  ]
}
\`\`\`

然后使用脚本导入：

\`\`\`typescript
import levels from '../data/word-levels.json';

for (const [level, words] of Object.entries(levels)) {
  await prisma.word.updateMany({
    where: { text: { in: words } },
    data: { level }
  });
}
\`\`\`

## 🔍 使用筛选功能

更新等级后，在搜索页面顶部会显示筛选按钮：
- **All**: 显示所有单词
- **IELTS**: 仅显示 IELTS 词汇
- **SAT**: 仅显示 SAT 词汇
- **GRE**: 仅显示 GRE 词汇
- **TOEFL**: 仅显示 TOEFL 词汇

## 💡 提示

1. 一个单词可能同时属于多个等级（如 IELTS 和 SAT）
2. 如果需要支持多标签，可以修改数据库 schema 使用数组字段
3. 建议从权威来源获取词汇列表以确保准确性
